- 这部分我以前推送到GitHub上项目已经创建过，可以参考我的文章Git



#### 1.3 提交

- 切换到提交目录

```
git init
git add -A
git commit -m 'deploy'

# 如果发布到 https://<USERNAME>.github.io
# git push -f git@github.com:<USERNAME>/<USERNAME>.github.io.git master
git push -f git@github.com:genghenggao/henggao.github.io.git master
```
